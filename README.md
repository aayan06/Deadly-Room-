# Deadly-Room-Twine
A suspenseful escape game where every choice matters. Find your way out… before it’s too late

🚪 Overview
Deadly Room Final is a psychological thriller built using Twine. Players wake up in a sterile, engineered environment where they must solve environmental puzzles involving symbols, floor patterns, and atmospheric pressure to unlock the exit.

🛠️ Built With
Engine: Twine 2.11.1

Story Format: Harlowe 3.3.9

Languages: HTML, CSS, JavaScript

🕹️ How to Play
Since this is a Twine game, it runs directly in your web browser.

Download the Deadly_Room_Final.html file from this repository.

Open the file using any modern web browser (Chrome, Firefox, Safari, or Edge).

Alternatively, if GitHub Pages is enabled, you can play it at: https://yourusername.github.io/deadly-room-twine/

🧠 Mechanics
Mistake Tracking: A hidden $mistakes counter monitors failed interactions. Too many errors lead to a "Game Over" state.

Environmental Puzzles: Players must cross-reference symbols on the North Wall with keypad inputs.

Pressure System: Waiting too long or idling in certain passages increases the $pressure variable, affecting the game's narrative tension.
